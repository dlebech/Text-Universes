About Text Universes

How to use

Click and drag anywhere on the screen to navigate around in the universe.

To start your own text universe, simply type in the name in the address bar. For example: /mycooluniverse. To add to the main universe, use /main.

You can either type in text yourself or make a Twitter search on keywords (there's a button for each of those). With Twitter search, the latest Twitter results for the given keywords are automatically added to the universe.

Future

The goal of Text Universes is to have a Text Mining algorithm cluster similar pieces of text together but this is not implemented yet.

Other than that, there are several other features which could be fun to have. But Text Mining is the core idea so it's rather strange it hasn't been done yet, right?

People

It will probably be completely open sourced pretty soon. Stay tuned.

Background

Text Universes is inspired by your world of text. Text Universes is not a remake of this idea but a personal hobby experiment. You can still share any piece of text but Text Universes gives you slightly less freedom since you cannot choose where to put your text in the universe.
